GAP 2: Аня в стране чудес (Q108527991)

We decided to focus on his translations, which are a very important part of his literary output. After finding which of Nabokov's translations appear on Wikidata, we decided to focus on his translation of "Alice's Adventures in Wonderland". We noticed that it missed relevant information, namely publisher, place of publication, and illustrator. We used LLM to find the missing information, and to create RDF triples representing that information.

QUERIES

Description of the Query SPARQL Query
We ran a query to see all items with the property “translator” (P655) and object “Vladimir Nabokov” (Q36591). 
SELECT ?item ?itemLabel
WHERE { 
  ?item wdt:P655 wd:Q36591
  SERVICE wikibase:label {bd:serviceParam wikibase:language "en". } 
}
We ran a query to retrieve information about “place of publication” (P291), “illustrator” (P110) and “publisher” (P123) for all items that have the property “edition or translation of” (P629) and value “Alice’s Adventures in Wonderland” (Q92640).
We used the keyword OPTIONAL so the results would include items that didn’t satisfy the requests about publisher, place, and illustrator. 
SELECT ?item ?itemLabel ?publisher ?publisherLabel ?placeofpublication ?placeofpublicationLabel ?illustrator ?illustratorLabel
WHERE {
  ?item wdt:P629 wd:Q92640 .   
  OPTIONAL { ?item wdt:P123 ?publisher .}  
  OPTIONAL { ?item wdt:P291 ?placeofpublication .}  
  OPTIONAL { ?item wdt:P110 ?illustrator .}  
  SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". }
}
ORDER BY DESC (?itemLabel)
LIMIT 50
We encountered another problem: the publisher, Gamaiun, doesn’t have a Wikidata profile. We ran a query to confirm this. 
SELECT ?item ?itemLabel 
WHERE {
  ?item wdt:P31 wd:Q2085381 .        
  ?item rdfs:label ?label .           
  FILTER(REGEX(?label, "gamaiun", "i"))
  SERVICE wikibase:label { bd:serviceParam wikibase:language "en". }
}
LIMIT 50
To be sure, we asked an LLM to generate a query that would find any items that match the string “gamaiun”.
None of the results were publishing houses. 
SELECT ?item ?itemLabel WHERE {
  SERVICE wikibase:mwapi {
    bd:serviceParam wikibase:endpoint "www.wikidata.org" .
    bd:serviceParam wikibase:api "EntitySearch" .
    bd:serviceParam mwapi:search "gamaiun" .
    bd:serviceParam mwapi:language "en" .
    ?item wikibase:apiOutputItem mwapi:item .
  }
  SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". }
}
LIMIT 50

LLM PROMPTING

Both gave us correct and synthetic information regarding the place of publication and the publisher, but two different answers regarding the illustrator. We checked an external source (Wikipedia), and found that the correct answer is Serge Chubine Q4185589 (alias Sergei Zalshupin). This is an example of an AI hallucination.
